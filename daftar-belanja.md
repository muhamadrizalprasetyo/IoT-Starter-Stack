*Beli ini supaya proyekmu bisa membaca lingkungan:*
* **DHT11 / DHT22**: Untuk baca suhu dan kelembapan udara.
* **HC-SR04**: Sensor ultrasonik untuk ukur jarak (biasanya buat robot).

* **PIR Sensor (HC-SR501)**: Sensor gerak (biasanya buat alarm keamanan).
* **Potensiometer**: Untuk input putaran (seperti volume radio).

*Alat untuk mengeksekusi perintah:*

* **Resistor (220 ohm, 1k, 10k)**: WAJIB beli satu paket supaya komponen tidak terbakar.
* **Active/Passive Buzzer**: Untuk bunyi alarm/beep.
* **Relay 1 Channel / 2 Channel**: Supaya ESP32 kamu bisa menyalakan alat listrik rumah (Lampu 220V, Kipas, dll).
* **Servo SG90**: Motor kecil untuk penggerak (bisa buat buka kunci pintu otomatis).
