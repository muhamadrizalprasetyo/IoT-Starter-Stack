IDE (*Integrated Development Environment*) adalah software yang kita gunakan untuk menulis kode dan mengirimkannya (upload) ke perangkat IoT.

### 1. Arduino IDE (Paling Direkomendasikan)
Ini adalah software wajib bagi pemula. Sangat simpel dan dukungannya paling luas di dunia.
* **Kegunaan:** Menulis kode bahasa C++.
* **Kelebihan:** Sangat mudah digunakan, tinggal colok alat dan klik upload.
* **[Link Download Resmi](https://www.arduino.cc/en/software)** (Pilih versi Windows Win 10 and newer).

### 2. VS Code + PlatformIO (Level Menengah)
Jika kamu merasa Arduino IDE terlalu sederhana, gunakan ini.
* **Kegunaan:** Menulis kode C++ dengan fitur *auto-complete* (saran kode otomatis).
* **Kelebihan:** Lebih cepat dan terlihat sangat profesional.
* **[Link Download VS Code](https://code.visualstudio.com/)**

### 3. Thonny IDE (Khusus Python)
Jika kamu tidak suka C++ dan ingin mencoba **MicroPython**, ini softwarenya.
* **Kegunaan:** Menulis kode Python untuk mikrokontroler.
* **Kelebihan:** Sangat ringan dan tidak membingungkan.
* **[Link Download Thonny](https://thonny.org/)**
